# solo-foss-the-hard-way

A manifest for not office workers, but for
solo self-taught developers who tired of how hard it became
to find an information for knowledge.

Firstly it will contain dumb chat-style talks
between three persons. Then I'll be refactoring
into structures, expanding them. It's for low 
cognitive load for me.

It will be first a collection of different decisions
so I can collapse them into a meaningful read later.
